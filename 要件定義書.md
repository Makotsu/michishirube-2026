# ミチシルベ2026 セッション・登壇者ページ 要件定義書

**作成日**: 2026年2月2日
**更新日**: 2026年2月2日
**バージョン**: 2.0

---

## 1. プロジェクト概要

### 1.1 目的
ミチシルベ2026の公式サイト（https://michishirube.okinawa/）からの遷移先として、セッションと登壇者情報をわかりやすくまとめたWebサイトを作成する。**本家サイトとデザインの統一感を持たせ、ユーザーがシームレスに移動できる体験を提供する。**

### 1.2 ターゲットユーザー
- イベント参加予定者
- セッション選択を検討中の方
- 登壇者について詳しく知りたい方
- イベントに興味を持った一般の方

### 1.3 本家サイト情報
- **イベント名**: ミチシルベ2026
- **キャッチフレーズ**: 「集い、響きあい、踏み出す。心地よい未来へ。」
- **概要**: 心と体の全部で、沖縄のいまと未来を感じることができる複合型のイベント
- **URL**: https://michishirube.okinawa/

## 2. デザイン要件（本家サイト準拠）

### 2.1 カラーパレット

#### テーマカラー（7色）
| 名称 | カラーコード | 用途 |
|------|-------------|------|
| Sky（空） | `#74b5e4` | 明るいアクセント、ホバー状態 |
| Sea（海） | `#427bbf` | プライマリカラー、ヘッダー、ボタン |
| Land（大地） | `#6cbc64` | 成功表示、自然系コンテンツ |
| Culture（文化） | `#eca25d` | アクセント、CTA |
| Heart（心） | `#e6786a` | ハイライト、重要情報 |
| Star（星） | `#f0ea30` | 注目情報、バッジ |
| Spirit（精神） | `#b077b0` | 特別セッション、装飾 |

#### ベースカラー
| 用途 | カラーコード |
|------|-------------|
| 背景（白） | `#ffffff` |
| 背景（薄グレー） | `#f8f9fa` |
| テキスト | `#333333` |
| サブテキスト | `#666666` |
| ボーダー | `#dee2e6` |

### 2.2 タイポグラフィ

| 用途 | フォント |
|------|----------|
| 見出し（和文） | A1明朝 → Noto Serif JP → Yu Mincho |
| 見出し（英文） | EB Garamond |
| 本文（和文） | Tsukushi Gothic → Noto Sans JP |
| 本文（英文） | Inter |

### 2.3 UIスタイル
- **ボタン**: 角丸40px（丸みを帯びた形状）
- **カード**: 角丸16px、柔らかいシャドウ
- **トランジション**: `.4s cubic-bezier(.4,.4,0,1)`
- **ホバー**: Y軸移動 + シャドウ強化
